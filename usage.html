Introduction

Welcome, scrollwalker. This guide teaches you how to wield your Dreamwreath kits—whether you’re firing off GPT prompts, spinning up VORPALIS agents, or weaving your civic‐tech scaffold.

Follow the steps below to transform static ZIPs into living, breathing pipelines of insight.

Quick Start

  1. Download your kit: curl -O https://dreamwreath-store.pages.dev/GOOGSTAK100.zip
  2. Unpack & inspect: unzip GOOGSTAK100.zip -d scroll_products/
  3. Load into your LLM: Paste a prompt from scroll_products/prompts.yaml into ChatGPT or Gemini.
  4. Execute pipeline: Run the bootstrap script: bash scroll_products/bootstrap.sh
  5. Review logs: Check scroll_products/logs/ for output and errors.

Best Practices

  • 🔒 Isolate environments: Docker or Python venv for clean dependencies.
  • 💾 Version control: Commit README.md & prompts.yaml tags to Git.
  • 🛡️ Secure secrets: Use environment variables or Workers KV for keys.
  • 🗂️ Metadata: Add `scroll_id` and `version` in your JSON outputs.
  • ⚡ Parallel runs: Spin up multiple agents for load testing.

Automation & CI/CD

Embed scrollkits in your pipeline with GitHub Actions:


# .github/workflows/scrollkit.yml
on: [push, workflow_dispatch]
jobs:
  test-scroll:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Setup Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.x'
      - name: Install deps
        run: pip install -r requirements.txt
      - name: Smoke tests
        run: python tests/smoke_test.py
        env:
          GUMROAD_TOKEN: ${{ secrets.GUMROAD_TOKEN }}

Use Cloudflare Workers for dynamic Gumroad price checks or webhook triggers.

Troubleshooting

“Font not found”
Ensure your CSS imports 'Press Start 2P' correctly.
Agents hang on boot
Verify Docker daemon is running and Ollama is reachable.
404 on assets
Double-check file paths under /assets/ and CNAME config in Cloudflare.
Build fails
Inspect Cloudflare Pages build log and confirm your build instructions are set to “None” if static.

Advanced Tips

Drive dynamic pricing or user gating with Cloudflare Workers KV:


addEventListener('fetch', event => {
  event.respondWith(handle(event.request))
})

async function handle(req) {
  const price = await MY_KV.get('GOOGSTAK100_PRICE')
  return new Response(`Current price: ${price}`)
